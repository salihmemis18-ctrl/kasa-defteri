Mevcut Firebase + HTML uygulamamı profesyonel SaaS sistemine çevir. Aşağıdaki özellikleri eksiksiz ve hatasız şekilde ekle: 1. Firma Bazlı Sistem: * Her kullanıcı bir firmaya bağlı olsun * Firestore yapısı: kullanicilar/{uid} → firmaId firmalar/{firmaId} → tüm veriler burada tutulacak * Tüm veri işlemleri (okuma/yazma) firmaId üzerinden yapılsın 2. Çoklu Kullanıcı: * Aynı firmaId ile birden fazla kullanıcı sisteme girebilsin * Aynı firmaya bağlı kullanıcılar aynı veriyi görsün 3. Admin Panel: * Sadece admin rolüne sahip kullanıcı erişebilsin * Yeni kullanıcı oluşturabilsin * Kullanıcıyı bir firmaya bağlayabilsin (firmaId seçilebilsin) * Kullanıcı listesi görüntülensin 4. Rol Sistemi: * kullanicilar koleksiyonuna rol alanı ekle: * admin * user * Admin panel sadece admin’e açık olsun 5. İlk Giriş Şifre Değiştirme: * Yeni oluşturulan kullanıcılar ilk girişte şifre değiştirmek zorunda olsun 6. Güvenlik: * Firestore kurallarını firmaId bazlı yap: Kullanıcı sadece kendi firmasının verisini okuyabilsin 7. Kod Yapısı: * Firebase Auth + Firestore düzgün entegre olsun * onAuthStateChanged ile kullanıcı kontrolü yapılsın * Loading ekranı doğru yönetilsin (takılma olmasın) Tüm kodu eksiksiz ve çalışır şekilde ver.
